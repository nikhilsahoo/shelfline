# Screenshot Assets

The README expects PNG screenshots at these exact paths:

- `docs/assets/screenshots/catalog-books.png`: OPDS catalog book listing from
  Calibre-Web, with visible authors and a selected book row.
- `docs/assets/screenshots/catalogs.png`: saved catalog selector with the
  catalog detail pane visible.
- `docs/assets/screenshots/catalog-folders.png`: OPDS folder or group
  navigation view.
- `docs/assets/screenshots/library.png`: local library view with the detail
  pane visible.
- `docs/assets/screenshots/reader.png`: EPUB reader view.

0.2.0 screenshot capture checklist:

- Catalog detail view showing either terminal image cover rendering or the
  polished text cover fallback.
- Library detail view showing cached cover metadata for a downloaded book.
- Normal EPUB reader view with configured paragraph spacing visible.
- Zen Mode EPUB reader view.
- Reader options overlay or bookmark navigator.

Suggested capture size: about 1200x800 pixels, or a terminal window large enough
to show the main list, status area, and detail pane without wrapping important
labels. Keep all images as PNG files.

Before adding screenshots, review them for private data. Redact or replace local
book paths, hostnames, usernames, access tokens, private catalog URLs, and book
titles or author names that should not be public.
